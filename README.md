# Bluestock Mutual Fund Analytics Capstone Project

End-to-end data pipeline, analytics platform, and interactive dashboard for analyzing 40 mutual fund schemes over a 3-year period. Powered by a **Neon DB (PostgreSQL)** backend and a **FastAPI + Streamlit + React** tech stack.

## Project Structure

```
├── auth_gateway/       FastAPI auth backend (JWT, Google OAuth, password reset)
├── dashboard/          Streamlit multi-page analytics dashboard
├── mainframe-landing/  React + Vite marketing landing page
├── scripts/            Python ETL, metrics, recommender, notebook generation
├── sql/                PostgreSQL schema and analytical queries
├── data/
│   ├── raw/            Raw CSV dataset files
│   └── processed/      Cleaned CSV files (generated by ETL pipeline)
├── notebooks/          Auto-generated Jupyter Notebooks (EDA, Performance, Advanced)
├── reports/            Generated visualizations and markdown reports
├── tests/              pytest test suite with deterministic fixtures
└── requirements.txt    Python dependencies
```

## Prerequisites

- Python 3.11+
- Node.js 18+ (for the React landing page)
- A [Neon DB](https://neon.tech) PostgreSQL database

## Setup Instructions

### 1. Install Python Dependencies

```bash
pip install -r requirements.txt
```

> **Important:** `psycopg2-binary` is required for PostgreSQL connectivity and is included in `requirements.txt`.

### 2. Configure Environment Variables

Copy and fill in the root `.env` file:

```bash
# Already present — update the placeholders:
ENVIRONMENT=production
SECRET_KEY=<generate with: python -c "import secrets; print(secrets.token_hex(32))">

# Neon DB (PostgreSQL) — from your Neon console
DB_URI_POOLED=postgresql://...
DB_URI_NON_POOLED=postgresql://...

# Deployment URLs
AUTH_BASE_URL=http://localhost:8000   # Your FastAPI domain in production
STREAMLIT_URL=http://localhost:8501   # Your Streamlit domain in production

# CORS — comma-separated origins allowed to call the API
ALLOWED_ORIGINS=http://localhost:8501,http://localhost:5173

# SMTP Email (for password reset emails)
# Gmail: enable App Passwords at https://myaccount.google.com/apppasswords
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USERNAME=your-email@gmail.com
SMTP_PASSWORD=your-app-password
SMTP_FROM_EMAIL=your-email@gmail.com

# Google OAuth
GOOGLE_CLIENT_ID=...
GOOGLE_CLIENT_SECRET=...
```

### 3. Initialize the Database Tables

Run this once to create all tables in your Neon DB:

```bash
python auth_gateway/init_db.py
```

### 4. Run the Data Pipeline

```bash
python run_pipeline.py              # Run all phases
python run_pipeline.py --phase 1   # Phase 1: Live NAV fetch + ingestion diagnostics
python run_pipeline.py --phase 2   # Phase 2: ETL + load to Neon DB (PostgreSQL)
python run_pipeline.py --phase 3   # Phase 3: Recommender demo
python run_pipeline.py --phase 4   # Phase 4: Generate EDA + analytics notebooks
```

### 5. Start the Auth Gateway (FastAPI)

```bash
uvicorn auth_gateway.main:app --host 0.0.0.0 --port 8000 --reload
```

### 6. Launch the Analytics Dashboard (Streamlit)

```bash
streamlit run dashboard/app.py
```

### 7. Start the Landing Page (React + Vite)

```bash
cd mainframe-landing
cp .env.example .env          # Set VITE_AUTH_URL to your FastAPI URL
npm install
npm run dev
```

## Features

- **Robust ETL**: Handles inconsistent date formatting and structurally varied datasets using `pandas` and `SQLAlchemy` with Neon DB (PostgreSQL).
- **Advanced Metrics**: Daily returns, 1/3/5-Year CAGR, Sharpe Ratio, Sortino Ratio, Alpha, Beta, Information Ratio, Upside Capture, VaR (95%), CVaR, Maximum Drawdown, and Herfindahl-Hirschman Index (HHI).
- **Automated Reporting**: Python scripts auto-generate rich interactive Jupyter Notebooks and charts using `nbformat`.
- **Interactive Dashboard**: A 6-page Streamlit app with Market Overview, Fund Performance & Risk, Investor Demographics, Portfolio Holdings, Fund Scorecard & Rankings, and Smart Fund Recommender.
- **Fund Scorecard & Recommender**: A custom recommendation engine driven by a weighted composite score of financial fundamentals, matching users to top-tier funds based on risk appetite.
- **Authentication**: Full JWT-based auth with Google OAuth, bcrypt password hashing, rate limiting, and real SMTP email for password resets.
- **Live NAV**: Fetches real-time NAV data from mfapi.in for 6 key mutual fund schemes.

| Dashboard | Streamlit + Plotly |

| Data Pipeline | pandas + SQLAlchemy |
| Analytics | scipy + numpy |
| Email | smtplib (SMTP/TLS) |
| Testing | pytest |
