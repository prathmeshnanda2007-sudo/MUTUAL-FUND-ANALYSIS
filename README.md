# Bluestock Mutual Fund Analytics Capstone Project

This repository contains the end-to-end data pipeline, analytics, and interactive dashboard for the Bluestock Mutual Fund Capstone Project. It analyzes data for 40 mutual fund schemes over a 3-year period.

## Project Structure
- `data/`
  - `raw/`: Raw CSV data files.
  - `processed/`: Cleaned and standardized CSV data files.
  - `db/`: SQLite database `bluestock_mf.db`.
- `scripts/`: Python scripts for data ingestion, ETL, metric computation, and notebook generation.
- `sql/`: SQL schema and analytical queries.
- `notebooks/`: Jupyter Notebooks generated by the pipeline for EDA, Performance Analytics, and Advanced Risk Metrics.
- `reports/`: Generated visualizations and markdown reports.
- `dashboard/`: Streamlit dashboard code.

## Setup Instructions

1. **Install Dependencies:**
   ```bash
   pip install pandas numpy sqlite3 matplotlib seaborn plotly nbformat streamlit scipy
   ```

2. **Run the Pipeline:**
   The entire pipeline can be executed end-to-end:
   ```bash
   python run_pipeline.py
   ```
   Or, run specific phases:
   ```bash
   python run_pipeline.py --phase 1  # Data Ingestion
   python run_pipeline.py --phase 2  # ETL Pipeline & SQLite Database Loading
   python run_pipeline.py --phase 3  # Mutual Fund Recommender Demo
   python run_pipeline.py --phase 4  # Analytics Generation (EDA, Performance, Advanced)
   ```

3. **Launch the Dashboard:**
   To explore the visual dashboard:
   ```bash
   streamlit run dashboard/app.py
   ```

## Features

- **Robust ETL**: Handled inconsistent date formatting and structurally varied datasets using `pandas` and `SQLAlchemy`.
- **Advanced Metrics**: Daily returns, 3-Year CAGR, Sharpe Ratio, Sortino Ratio, Alpha, Beta, VaR, CVaR, and Herfindahl-Hirschman Index (HHI).
- **Automated Reporting**: Python scripts auto-generate rich interactive Jupyter Notebooks and charts using `nbformat`.
- **Interactive Dashboard**: A multi-page Streamlit app providing high-level overviews, dynamic performance comparisons, demographic breakdowns, and macro-level industry trends.
- **Fund Scorecard & Recommender**: A custom recommendation engine driven by calculated financial fundamentals, matching users to top-tier funds based on their risk appetite.
